Write a simple palindrome function (less than 160 characters) that returns a boolean indicating whether or not a string.

           //replace() method searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced
            //toLowerCase method to covert all character into lower case

           function isPalindrome(str) {
            if(str.length<160 || typeof (str) == 'string'){ //check string is valid string and length is less than 160 char
            str = str.replace(/\W/g, '').toLowerCase();
            return (str == str.split('').reverse().join(''));
            }else{
              return false;
            }
          }
